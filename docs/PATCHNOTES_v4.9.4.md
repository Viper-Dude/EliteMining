**Release Date:** 2026-03-16

## New Features & Improvements

### Backup & Restore
- Added Configuration (config.json) as a backup option — saves all your settings and presets
- Configuration can now be restored from backup as well
- Bookmarks now refresh immediately after restoring from backup — no restart needed
- Restore dialog is now slightly taller to fit the new option

### Installer
- Added optional checkboxes for installing EliteVA (EliteAPI) and VoiceAttack profile
- EliteVA install is skipped automatically if the same version is already installed
- VoiceAttack profile checkbox now auto-checks when a newer profile version is bundled
- If the bundled profile matches what's already installed, the checkbox stays unchecked
- Renamed "voice automation" label to "VoiceAttack plugin" for EliteVA

### UI Fixes
- Backup and Restore dialogs no longer hide behind the main window



### Voiceattack Profile
- No changes - latest version is v4.8.3 (included in the installer and zip file)
