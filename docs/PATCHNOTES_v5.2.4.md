## Voiceattack v5.2.4
* Fixed an issue where the command to change the fire group from **H** to **Fire Group C** was not working.


* Renamed the "Core Mining" section in the VoiceAttack Controls tab to "General Timer", since its timer is now also used for the FSD Jump Sequence and the Continuous Boost command.

* Added additional SRV commands.

* FSD Jump Sequence – You can now control how long the system map stays open, using a timer in the VoiceAttack Controls tab under "General Timer".
