## New Features & Improvements



### Voiceattack Profile - Updated to v5.0.0S

- Prospector sequence: Increased the thrust up timer from 1 to 1.2 seconds when launching a prospector
- Added option to preserve custom keybinds for updating the VoiceAttack profile — available via Settings

### VoiceAttack Controls Tab

- Added **Thrust up duration** spinbox (1.0–2.0s, default 1.2s) — controls how long the upward thrust is applied during the prospector sequence. Value saved as part of the ship preset.


#### Important Notes 

- Remember to always check all your key and joystick bindings after every profile update.
