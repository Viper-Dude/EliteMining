# <img src="https://github.com/Viper-Dude/EliteMining/blob/main/app/Images/logo_multi.ico" width="32" height="32"> EliteMining v4.6.5
**Release Date:** November 30, 2025

## New Features & Improvements

### Mining Session
- **Edit Hits Count** - Double-click on the Hits column in Mineral Analysis to correct accidental double prospects

### Bookmarks
- **Rating System** - New Rating column (1-5 stars) to rate your favorite mining locations

### Ring Finder
- **Visits Column** - The "Visited" column now shows actual visit count instead of Yes/No, renamed to "Visits"
- **Visit Count Fix** - One-time database migration corrects inflated visit counts (only counts FSD jumps, not game loads)

### General
- **Total Systems Visited** - Added "Tot Syst" counter to the CMDR status bar showing lifetime systems visited
- UI polish and theme consistency improvements

## Bug Fixes
- Fixed auto-switch to Ring Finder tab not working on FSD jump

### Voiceattack Profile
- No changes - latest version is v4.6.0 (included in the installer and zip file)

## Notes!
- If installer fails with errors, right-click the installer and select "Run as administrator"
- First launch after update will run a quick database migration to fix visit counts
