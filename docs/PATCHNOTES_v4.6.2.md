# <img src="https://github.com/Viper-Dude/EliteMining/blob/main/app/Images/logo_multi.ico" width="32" height="32"> EliteMining v4.6.2
**Release Date:** November 28, 2025

## New Features & Improvements

### Theme Options
- Added theme toggle button in the bottom status bar
- Choose between **Dark Gray** theme or **Elite Orange** theme (classic ED HUD style)
- Theme preference is saved and remembered

### UI Improvements
- Smaller Import/Apply buttons for a cleaner look
- Improved button styling throughout the app
- Alternating row colors in Announcement Panel for better readability
- Better default window size for new installations
- Elite Orange theme now applies correctly to Tips/Help section

### Hotspots Finder
- Removed the Density column (data was unreliable)

### VoiceAttack Profile
- No changes - latest version is v4.6.0 (included in the installer and zip file)

## Notes!
- Theme change requires app restart to fully apply
- If installer fails with errors, right-click the installer and select "Run as administrator"
