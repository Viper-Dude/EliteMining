# <img src="https://github.com/Viper-Dude/EliteMining/blob/main/app/Images/logo_multi.ico" width="32" height="32"> EliteMining
**Release Date:** 2025-01-31

## New Features & Improvements

### VoiceAttack Controls Tab
- **New: Prospector Cooldown Timer** - Adjustable timing (2.0-6.0s, default 4.4s)
- **New: Thrust Up Timing Controls** - Separate controls for scoop retracted (1.5s) and deployed (3.8s)
- **UI Improvements**: Cleaner interface, removed redundant "Enable" prefixes, improved help text

### Hotspot Finder
- Added option for manual set Reserve Level (right-click context menu)
- Added "Reset Search Filters" button to quickly reset all search filters to defaults

### Bug Fixes & Improvements
- **Fixed:** Keybind preservation now correctly handles compound keybinds (multiple buttons combined with `+`)
- **Fixed:** Profile update dialog no longer hides behind main window
- **Fixed:** Thrust Up timing now uses correct settings for cargo scoop retracted/deployed

### Voiceattack Profile
- Updated to v4.8.5 (included in installer)
- **Ship Presets**: Now managed exclusively via EliteMining App UI (removed from VoiceAttack)
