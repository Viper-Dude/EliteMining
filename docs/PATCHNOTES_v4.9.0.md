**Release Date:** 2025-01-30

## New Features & Improvements

- Added **Kestrel Mk II** ship support
- New Ship Preset dialog now pre-selects your current ship
- Faster app startup (journal scan optimized)
- Fixed: Ship Type dropdown in New Ship Preset was hidden behind the dialog
- Fixed: New Ship Preset dialog could appear behind the main window
- **Reports tab**: Filters split into 3 combinable dropdowns (Time, Performance, Source)
  - Time: All Time, 1d, 2d, 3d, 7d, 30d, 90d
  - Performance: High/Medium/Low Yield, Hit Rate, 50+ Tons, Core Mining
  - Source: Platinum, High-Value, Common, With Engineering Mats, By System, By Ship
  - Filters can be combined (e.g. "Last 7 days" + "High Yield" + "Platinum")
  - By System / By Ship shows a second dropdown with your session history
